WHAT IS THIS?

Speedgun.js is a simple PhantomJS 2 based script that collects performance metrics from a single web page. This site allows you to point Speedgun at any public facing website and gather performance numbers.

GETTING STARTED

Speedgun.js is extremely simple to setup and use. It is one file and only requires you to have PhantomJS 2 installed.

  • Clone the repository (or curl the file)
  • Run it
  • View the report

You can also run this same server via Docker. It's all OSS, so feel free to stand up your own instance. The more Speedgun servers running in the world, the better our web performance will be.

Mac or Windows Only

  • Clone the repository (git clone ...speedgun)
  • Install vagrant
  • cd speedgun/server
  • Start vagrant and ssh into it (vagrant up && vagrant ssh)
  • Build the container (cd speedgun && docker build .)
  • Run the container (docker run -ti -p 8080 --net=host [latest image ID] /bin/bash (get the image ID from 'sudo docker ps -a')
  • Open browser go to http://localhost:8081

Linux

  • Clone the repository (git clone ...speedgun)
  • Install Docker (i.e. debian/centos: sudo yum -y install docker-io)
  • Build the container (cd speedgun && docker build .)
  • Run the container (docker run -ti -p 8080 --net=host [latest image ID] /bin/bash (get the image ID from 'sudo docker ps -a')
  • Open browser go to http://localhost:8080
