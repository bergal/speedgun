WHAT IS THIS?

Speedgun.js is a simple PhantomJS 2 based script that collects performance metrics from a single web page. This site allows you to point Speedgun at any public facing website to gather important web performance data.

GETTING STARTED

Speedgun.js is extremely simple to use. It is one file and only requires you to have PhantomJS 2 installed.

  • Clone it: git clone https://github.com/wesleyhales/speedgun.git
  • Run it: cd speedgun && phantomjs --ssl-protocol=any --ignore-ssl-errors=yes core/speedgun.js http://www.wesleyhales.com performance csv
  • View the report in speedgun/core/reports
  • View the Readme for more options.

You can also run the speedgun.io server via Docker. It's all OSS, so feel free to stand up your own instance. The more Speedgun servers running in the world, the better off web performance will be.

Mac or Windows

  • Clone it: git clone https://github.com/wesleyhales/speedgun.git
  • Install Vagrant
  • cd speedgun/server
  • Start vagrant and ssh into it: vagrant up && vagrant ssh
  • Build the container: cd /vagrant && docker build .
  • Run the container: docker run -ti -p 8080 --net=host [latest image ID] /bin/bash (get the image ID from sudo docker ps -a)
  • Open browser go to http://localhost:8081

Linux

  • Clone it: git clone https://github.com/wesleyhales/speedgun.git
  • Install Docker (i.e. for centos: sudo yum -y install docker-io)
  • Build the container: cd /vagrant && docker build .
  • Run the container: docker run -ti -p 8080 --net=host [latest image ID] /bin/bash (get the image ID from sudo docker ps -a)
  • Open browser go to http://localhost:8080
